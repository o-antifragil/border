Bordas

O que são e para o que servem as bordas?

As bordas em CSS são usadas para definir uma linha ao redor de elementos, ajudando a separar, destacar ou organizar conteúdos na página. A propriedade básica para bordas é border, que engloba diversos aspectos, como largura, estilo e cor. Com ela, você pode criar bordas sólidas, tracejadas, arredondadas, coloridas e muito mais.

Principais Propriedades de border

  1. border-width: Define a espessura da borda, que pode ser especificada em unidades como px ou usando palavras-chave como thin, medium, thick. 
    border-width: 2px;
  2. border-style: Define o estilo da borda, como solid (sólido), dashed (tracejado), dotted (pontilhado), entre outros. 
    border-style: solid;
  3. border-color: Define a cor da borda, podendo ser qualquer cor válida em CSS. 
    border-color: #333;

Essas propriedades podem ser combinadas em uma única linha usando a abreviação border.

.elemento {
    border: 2px solid #333;
}

Propriedades Individuais para Cada Lado

Você pode aplicar bordas diferentes em cada lado do elemento com propriedades como:

  • border-top
  • border-right
  • border-bottom
  • border-left

Cada uma pode ter seus próprios valores de largura, estilo e cor.

.elemento {
    border-top: 2px solid red;
    border-right: 3px dashed blue;
}

border-radius: Bordas arredondadas

Para criar bordas arredondadas, usamos border-radius, que permite arredondar os cantos do elemento. Um valor maior de border-radius cria um círculo ou elipse. 

.elemento {
    border: 2px solid black;
    border-radius: 10px;
}

Resumo de Propriedades

  • border: Define todos os aspectos de uma borda de uma só vez.
  • border-width, border-style, border-color: Definem largura, estilo e cor de forma independente.
  • border-radius: Arredonda os cantos da borda.

As bordas são muito úteis para adicionar design e estrutura ao layout da página.
